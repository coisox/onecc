  • W1CC v1.5.2 & M1CC v1.2.2 20 July 2017
    Whats New
    • BOTH: New defination for:
      • Active Callcard = A callcard which hasn't been classified regardless the number of responders and their status
      • History Callcard = A callcard which has been classified regardless the number of responders and their status
      • Assigned Callcard = A callcard which has at least 1 responder regardless it has been classified or not
      • Unassigned Callcard = A callcard which has 0 responder, or none of the responders has been dispatched regardless the Callcard has been classified or not
    • BOTH: After a Callcard has been classified, these actions will be disabled:
      • Block new chatting. Existing chatting can still be accessed.
      • Block manage responder
      • Block change responder status
    • BOTH: Chatroom will auto launch on mobile only not in W1CC
    • W1CC: Change mini stat to reflect Callcard List module
    • W1CC: Default filter in Callcard List module is 'active'
    • W1CC: In Callcard List module, add new filter 'Unassigned'
    • M1CC: After a Callcard has been classified, add photo will be disabled
